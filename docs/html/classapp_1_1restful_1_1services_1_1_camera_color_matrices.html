Detailed Description

Camera color matrices for RAW-to-sRGB conversion.

These 3×3 matrices are camera-specific transformations from camera RGB (post-demosaicing) to standard sRGB color space. Values derived from dcraw's adobe_coeff table and calibrated against color charts.

Matrix application: [R', G', B'] = M × [R, G, B]

Source: LibRaw/dcraw color coefficient database
