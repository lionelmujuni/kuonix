Detailed Description

Wires up LangChain4j infrastructure beans:

  • ChatMemoryProvider — creates a per-session sliding-window memory (max 10 messages) keyed by the sessionId passed to KuonixAiService.chat()
  • ChatModelListener — logs token usage for every LLM call so cost and latency can be monitored

These beans are always created. The AI service itself is conditional based on user settings (managed via DynamicOllamaConfig).
