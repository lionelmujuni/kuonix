TokenStream chat (@MemoryId String sessionId, @UserMessage String userMessage)
 

Detailed Description

Kuonix AI assistant powered by LangChain4j.

Bean is created manually by DynamicOllamaConfig only when a valid StreamingChatLanguageModel is available. This avoids startup failures when AI is disabled in user settings.

Session memory is keyed by sessionId so each Color Lab session gets its own independent conversation history.

Returns TokenStream so the controller can forward each token to the SSE stream as it arrives, giving the user real-time feedback.
