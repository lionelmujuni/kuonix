Detailed Description

Color correction algorithms based on photographic research. Implements White Balance (Gray World, White Patch, Shades of Gray), Exposure adjustment, Saturation enhancement, and Color Matrix transformation.

Reference: Bianco, S. (2010). Color Correction Algorithms for Digital Cameras. PhD Thesis.

◆ applyTemperatureTint()

Mat app.restful.services.ColorCorrectionService.applyTemperatureTint ( Mat bgr,
double tempK,
double tint )

Manual white balance using colour temperature (Kelvin) and tint (green↔magenta).

Converts Kelvin to approximate RGB multipliers via the Planckian locus polynomial approximation (Kim et al. 2002), then applies a tint bias on the green channel. Multipliers are normalised so the brightest channel stays at 1.0.

Parameters
bgrsource image in BGR format (uint8)
tempKcolour temperature in Kelvin [2000–10000]
tintgreen/magenta bias [-1.0, +1.0]; positive = more green, negative = more magenta
