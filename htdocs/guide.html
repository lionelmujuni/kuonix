C. Color Lab (PRO)

The core editing suite where manual and algorithmic corrections happen. Color Lab implements 7 research-based color correction algorithms for professional photography workflows.

Screenshot: Color Lab Interface

Split-view preview with algorithm selection and parameter controls

Picture Preview

Large central viewing area with "Before/After" split-view toggle to compare original and corrected images in real-time.

7 Color Correction Algorithms

Choose the right algorithm for your specific correction needs:

1. Gray World

Assumes the average color of the scene should be neutral gray.

When to use: General white balance correction for mixed lighting scenes.

2. White Patch (Max RGB)

Assumes the brightest patch in the image should be pure white.

When to use: Images with clear white reference points (white walls, paper, clothing).

3. Shades of Gray

Advanced Minkowski p-norm method for sophisticated white balance.

When to use: Professional work requiring precise color accuracy (product photography, art reproduction).

4. Exposure Adjustment

Applies uniform gain across all color channels to fix under/overexposure.

When to use: Globally dark or bright images that need overall brightness correction.

5. Saturation Enhancement

HSV-based saturation control to boost or reduce color intensity.

When to use: Dull, flat images that need color vibrancy, or oversaturated images needing toning down.

6. Color Matrix Transform

3×3 linear transformation for color space conversion and creative color grading.

When to use: Advanced color grading, film emulation, or specific color space conversions.

⭐ 7. Color Distribution Alignment

Matches the color distribution of target images to a reference image using LAB color space analysis.

When to use: Multi-camera shoots (weddings, events), batch consistency, style transfer. See tutorial below.

Controls Panel

  • Algorithm Selection: Dropdown menu to choose from the 7 correction methods above.
  • Dynamic Adjustments: Parameter sliders appear based on selected algorithm (Correction Strength, Exposure Gain, Saturation Level, etc.).
  • Theory & Research: Each algorithm includes "Problem Solved" and "Algorithm Explanation" sections for understanding the correction approach.
  • Presets: Name and save current algorithm + parameter combinations for instant reuse.

Execution Options

  • Save This Image: Apply correction to the current image only
  • Apply to All: Batch process all images in the current set with the same settings
  • Undo All: Revert all changes back to original state

Grid view and single-image preview modes let you efficiently review results before final export.

🎯 Multi-Camera Consistency Tutorial

Use Color Distribution Alignment to ensure consistent look across images from different cameras or lighting conditions.

Workflow Steps:
1
Select Reference Image

Choose the image with the desired color and exposure as your reference. This could be from your main camera or the best-lit shot.

2
Load Target Images

Import all images that need color matching (from secondary cameras, different time of day, various lighting setups).

3
Select Color Distribution Alignment

Choose this algorithm from the dropdown. The app analyzes the reference image's LAB color space distribution.

4
Adjust Strength Parameter

Use the Correction Strength slider (0-100%) to control how closely target images match the reference. Start at 70-85%.

5
Preview & Compare

Use split-view to compare before/after. Check multiple images in grid view to ensure consistent results across the batch.

6
Apply to All

Click "Apply to All" to batch process all target images with the same reference-based correction.

💡 Perfect For:
  • Wedding Photography: Match images from multiple photographers/cameras
  • Event Coverage: Ensure consistent look across different lighting zones
  • Product Photography: Match new shots to existing catalog style
  • Film Emulation: Transfer color characteristics from reference film scans
