Workflow Model

What Kuonix is actually doing in the stack.

Kuonix is a platform-agnostic application used to sort images into categories based on color-correction needs such as contrast, white balance, exposure, and cast issues. It supports professional batch workflows by organizing work for tools like Photoshop, while also offering limited editing capabilities inside the app.

Staging

Bring in full shoots, preview what arrived, and remove unwanted frames before the engine starts classifying them.

Classification

Images are grouped by measurable technical defects, not by a vague aesthetic score or metadata assumption.

Correction

Color Lab provides limited but useful in-app correction tools for balancing, exposure alignment, saturation work, and reference matching.

Routing

Export opens folders in native tools so final creative work stays in the editor your team already uses.

Library

The Library view functions as a controlled review surface for everything that has been ingested. Operators can see thumbnails, issue labels, and batch-select images that need a common correction path.

Grid Review

Visual thumbnails keep the operator oriented while issue tags expose what the engine has already detected.

Selection Logic

Bulk selection tools make it practical to move large image groups into Color Lab or export subsets by defect type.

Action Surface

Delete, isolate, and send to correction workflows without bouncing between multiple tools or folders.

Issue Visibility

Classification stays attached to the image, which removes ambiguity when teams return to a job later.

Color Lab

Color Lab is the system’s limited editing bay. It exists for cases where classification has already identified a known technical problem and the operator wants a targeted in-app fix before pushing the batch into a primary editor.

Before
After

Gray World

Neutralizes white balance by assuming average scene color should approach gray.

White Patch

Uses the brightest region as a white reference when a clean highlight exists.

Shades of Gray

Extends white balance correction through a stronger p-norm approach for more demanding scenes.

Exposure Adjustment

Applies controlled global gain to repair frames that are uniformly dark or bright.

Saturation Control

Boosts or tamps down color intensity where the issue is global and obvious.

Color Matrix Transform

Supports deeper channel transforms for stylized grading and space conversion work.

Color Distribution Alignment

Matches target images to a reference look in LAB space, which is critical for consistency across multi-camera sets.

Operating Notes

Keep intake clean. The best automation results come from killing obvious misses before batch analysis starts.

Use issue classes as work queues. The taxonomy exists to reduce context switching during correction.

Reserve Color Lab for deliberate passes. It is strongest when paired with classification, not used blindly across every frame.

Keep finishing in native tools. Kuonix adds structure and speed; the final look still belongs in the editor.
